# 🥧 Bazaar Sales Analysis Dashboard

## 📌 Project Summary

This project analyzes transactional sales data from a 2-day baked goods bazaar event. The objective is to transform manually collected sales records into structured insights using Microsoft Excel.

The analysis focuses on revenue performance, product contribution, and customer distribution patterns.

---

## 📂 Dataset Overview

The dataset contains transaction-level records including:

- Date of Sale  
- Product Name  
- Transaction Amount  
- Customer Type (Student, Adult, Teenager, Elderly)

Each row represents a single sales transaction recorded during the event.

**Coverage:**
- 2 days
- Multiple product categories
- Real-world manually recorded data

---

## 🧹 Data Preparation

Before analysis, the dataset was:

- Cleaned to remove blank and summary rows
- Standardized for consistent price formatting
- Structured into an Excel Table
- Aggregated using Pivot Tables

This ensured accurate computation of revenue and customer distribution metrics.

---

## 📊 Analysis Performed

### 1️⃣ Daily Sales Performance
A line chart was created to visualize total revenue per day and identify the highest-performing day.

### 2️⃣ Product Revenue Comparison
A bar chart was used to compare total revenue generated by each product and determine best-selling items.

### 3️⃣ Customer Distribution Analysis
A pie chart was used to analyze purchase share by customer type.
